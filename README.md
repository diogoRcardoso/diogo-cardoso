## Gerador de Senhas Aleatórias – C#

Projeto em C# que gera senhas aleatórias com base no tamanho informado pelo usuário.

### Conceitos utilizados
- Entrada e validação de dados
- Estruturas de repetição
- Geração de números aleatórios
- StringBuilder para melhor desempenho

### Objetivo
Projeto educacional para prática de lógica de programação e boas práticas básicas em C#.
